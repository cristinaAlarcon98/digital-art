# Digital Art - p5.js Experiments

This project contains small **digital art experiments** created with **p5.js**.  
Each sketch explores shapes, colors, and motion using creative coding techniques.

## Description

Developed using **JavaScript**, **HTML**, and **CSS**, this project is part of my exploration into generative art and interactive visuals.

To run the project, simply open `index.html` in your browser.

## Files

- **`index.html`** – Base structure of the page.  
- **`sketch.js`** – Main p5.js code with drawing logic.  
- **`style.css`** – Basic styling for layout and visuals.  
- **`libraries/`** – p5.js library files.
